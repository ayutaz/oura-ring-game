🌐 GitHub Pages版について

このバージョンはブラウザのみで動作します。
データは localStorage に保存され、永続化されません。

🔗 Oura Ring連携設定

⚠️ 重要

GitHub Pages版では直接OAuthはできません。
Personal Access Tokenを使用して接続します。

📋 Personal Access Tokenの取得方法

  1. Oura Cloudにログイン
  2. Settings → Personal Access Tokens
  3. 「Create New Token」をクリック
  4. 必要なスコープを選択:
    • daily_readiness
    • daily_sleep
    • daily_activity
    • personal_info
  5. 生成されたトークンをコピー
